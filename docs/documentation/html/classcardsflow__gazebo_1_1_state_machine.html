enum cardsflow_gazebo::StateMachine::State

The state enum for different states of the state machine.

Differentiates between three possible states of the state machine: 0: Not Wrapping 1: Positive, for wrapping over angles smaller than 180° 2: Negative, for wrapping over angles larger than 180°
